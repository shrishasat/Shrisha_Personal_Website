MRI Preprocessing Project

Techniques Performed

  • Intensity Normalization
  • Skull Stripping
  • Talairach Transformation
  • Subcortical Segmentation
  • White Matter Segmentation
  • Cortical Surface Reconstruction
  • Cortical Parcellation
  • Cortical Thickness Estimation
  • Surface Registration

Pipeline: Environmental Setup

module load bear-apps/2022a
module load FreeSurfer/7.4.1-centos8_x86_64
module load dcm2niix
module load freesurfer

File Format Conversion

dcm2niix -o /Freesurfed_Anat/ \\ -f T1_{SubjectName}_volume \\ /AnatomicalMRIs/{SubjectFolder}/T1_vol_v1_5/

Reconstruction

recon-all -i /Freesurfed_Anat/T1_{SubjectName}_volume.nii \\ -s {SubjectName} –all

Monitor Progress/Log

cd /Freesurfed_Anat/subjects/{SubjectName}/scripts/
less recon-all.log

Estimated Processing Time: Approximately 4 hours per subject on 24 CPU cores.

Software Used

Freesurfer suite for structural MRI preprocessing and cortical reconstruction.
